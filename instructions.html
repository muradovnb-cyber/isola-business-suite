Как создать заказ

  1. Заказы → + Новый заказ в правом верхнем углу
  2. Введите название (например: «Кухня Ахмедов»), выберите клиента
  3. Укажите сумму договора и ответственного менеджера
  4. Нажмите Сохранить — заказ появится в списке

Как смотреть финансы

  1. Баланс счетов — реальные остатки: касса, банк, долги
  2. P&L — доходы и расходы по месяцам, маржа
  3. Маржинальность — прибыльность каждого заказа
💡 Telegram-бот (@isolainbot)Пишите в Telegram: «касса», «отчет», «смета Бухара» — бот ответит за секунды с реальными данными.

Управление сотрудниками

  1. Сотрудники — карточки всех сотрудников, подотчётные суммы
  2. Нажмите ✏️ Изменить на карточке — редактируйте данные, оклад, %
  3. Нажмите 🔑 Пароль — смените пароль для входа
  4. Управление — создание новых пользователей

Как добавить расход

  1. Перейдите в Транзакции → + Добавить расход
  2. Выберите дату, счёт (Касса / Банк / Долг) и валюту
  3. Выберите статью расхода и контрагента из списка
  4. Привяжите к заказу если это расход по конкретному проекту
  5. Нажмите Сохранить
💡 Быстрее через TelegramНапишите в группу «Расходы ISOLA»: «купил у BLUM 500к» — бот запишет автоматически.

Как начислить зарплату

  1. Зарплата → + Начислить
  2. Выберите сотрудника, месяц и год
  3. Проверьте сумму (система считает автоматически по окладу и %)
  4. Нажмите Начислить — запись появится в списке
  5. При выплате — нажмите Отметить оплаченным

Подотчётные средства

  1. Подотчёт — список всех выданных средств и их статус
  2. Статус открыт — средства ещё не отчитаны
  3. Статус отчитан — сотрудник предоставил подтверждение
  4. Статус закрыт — проверено и закрыто

Как работать с CRM

  1. CRM — Сделки → + Новая сделка
  2. Введите название сделки, выберите клиента из базы
  3. Укажите предполагаемую сумму и стадию: Переговоры
  4. По мере продвижения — меняйте стадию на Договор или Закрыта

Как создать заказ из сделки

  1. Когда сделка выигранна — перейдите в Мои заказы → + Новый заказ
  2. Привяжите к клиенту, укажите сумму договора
  3. Следите за прогресс-баром оплаты — зелёный = клиент оплатил полностью
💡 КомиссияПри закрытии заказа система автоматически рассчитает вашу комиссию по % от суммы.

Как записать расход по заказу

  1. Заказы → нажмите на нужный заказ
  2. В карточке заказа найдите блок «Добавить расход»
  3. Выберите дату, счёт (Касса/Банк/Долг) и статью расхода
  4. Укажите поставщика и сумму — нажмите Сохранить
💡 Ещё проще — через TelegramНапишите в группу «Расходы»: «купил у BLUM 500к для Ахмедов» — бот запишет сам и привяжет к нужному заказу.

Подотчётные средства

  1. Раздел Подотчёт — видите все выданные вам средства
  2. После расхода — добавьте запись с подтверждением (чек, фото)
  3. Нераспределённый остаток показывается в шапке карточки
⚠️ Не забывайте отчитываться по подотчётным суммам в конце поездки/закупки

Как работать с заказами

Заявки на услуги

  1. Заявки на услуги → + Новая заявка
  2. Выберите тип услуги (монтаж, сварка, покраска и т.д.)
  3. Привяжите к заказу и укажите желаемый срок выполнения
  4. Заявка уйдёт руководителю производства на согласование

Как вести заказы на производстве

  1. Раздел Заказы — список всех активных проектов
  2. Откройте заказ → добавляйте расходы по мере работы (материалы, услуги)
  3. Следите за статусом — меняйте на В работе → Готов
💡 Записывайте расходы сразуПроще всего написать в Telegram-группу: «купил краску 150к для Бухара» — бот всё запишет.

Заявки на услуги

  1. Заявки на услуги — список всех заявок от конструкторов
  2. Просматривайте детали: тип работы, заказ, срок
  3. Назначайте бригадиров и контролируйте выполнение
  4. Меняйте статус заявки: Новая → В работе → Выполнена

Подотчётные средства

  1. Раздел Подотчёт — ваши выданные средства
  2. После поездки или закупки — добавьте расходы с чеками
  3. Закройте подотчёт когда все суммы отчитаны
⚠️ Открытые подотчёты видны директору — отчитывайтесь своевременно
