# 🏢 ISOLA Business Suite

Корпоративный портал компании ISOLA — объединяет финансовый учёт и HR.

## Что включено
- 👑 **CEO Агент** — генератор идей, единый дашборд
- 💰 **FinApp** — P&L, баланс, маржинальность, заказы, транзакции
- 👥 **HR Portal** — CRM сделки, зарплата, профили сотрудников
- 🔗 **Интеграции** — CRM→Заказ, Выигрыш→Комиссия
